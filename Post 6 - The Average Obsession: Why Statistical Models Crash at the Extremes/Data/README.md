# ERA5 data for Post 6

The notebook `6.TheAverageObsession.ipynb` expects yearly NetCDF files in:

- `era5_surface/YYYY_surface_data.nc`
- `era5_pressure/YYYY.nc`

Coverage: **1996–2025**, Eastern Mediterranean coastal box (see notebook for exact lat/lon and variables).

If you clone the repo without these files, download matching ERA5 reanalysis extracts from [Copernicus CDS](https://cds.climate.copernicus.eu/) and place them in the folders above using the same naming pattern.
